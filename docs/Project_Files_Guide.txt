                                 distribution, extra visualizations)

  GRAPH 1 : model_comparison_chart.png
  --------------------------------------------------------------------------------
  Title    : Model Comparison — R² Score (5-Fold Cross-Validation)
  Location : diabetes+alzihmer\model_comparison_chart.png
  Use In   : IEEE Paper (Section IV — Results), PPT Slide 8, Zenodo Gallery

  Data Used (from model_comparison_results.txt):
  ┌──────────────────────┬────────┬──────────┬──────────┬──────────┐
  │ Model                │  R²    │ Adj R²   │  MAE     │  RMSE    │
  ├──────────────────────┼────────┼──────────┼──────────┼──────────┤
  │ Linear Regression    │ 0.8706 │  0.8593  │ 0.051602 │ 0.065723 │ ← BEST
  │ XGBoost              │ 0.8705 │  0.8592  │ 0.052658 │ 0.065913 │ ← MOST STABLE
  │ Random Forest        │ 0.8648 │  0.8530  │ 0.052869 │ 0.067196 │
  │ Decision Tree        │ 0.7436 │  0.7212  │ 0.073036 │ 0.092172 │
  └──────────────────────┴────────┴──────────┴──────────┴──────────┘
  All metrics = mean across 5-Fold CV (shuffle=True, seed=42, n=1000)

  Key Insight: Linear Regression and XGBoost nearly tied (R²=0.8706 vs 0.8705),
  but XGBoost showed lower variance (std=0.0084 vs 0.0178), making it more
  stable for deployment in clinical decision support systems.
